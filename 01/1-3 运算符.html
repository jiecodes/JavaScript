运算符(operator) 操作符 用于实现赋值、比较和执行算数运算等功能的符号 算数运算符 用于执行俩个变量或值的算术运算

+ - * / %(取余数 取模mu) 9 % 2 = 1 3 % 5 = 3 用途：余数是0 说明这个数能被整除 浮点数 在算数运算符里面会有问题 console.log(0.2 + 0.1) // 0.3000000000000004 console.log(0.7 * 100) // 0.7000000000000001 不能拿浮点数直接比较(原因：机器语言会转换为二进制 会有误差) 是否相等 var num = 0.1 + 0.2 console.log(num == 0.3 ); // false 算数运算符优先级 先乘除 后加减 有小括号先算小括号里面的

表达式 返回值

由数字 运算符 变量 组成的式子是表达式 1 + 1 = 2 程序里面的是 2 = 1 + 1 右赋值给左 1 + 1 算数表达式 2 返回值

前置自增运算符 ++递增 --递减

var num = 1; num = num + 1; num = num + 1; console.log(num); // 3 前置递增运算符 ++ var age = 10; ++age; // 类似于 age = age + 1 先加1 后返回值 var p = 10; console.log(++p + 10) // 21 后置递增运算符 num++; // 类似于 num = num + 1 先返回原值 后加1 先运算 后自增 先参加运算 得出结果后 再自增 var age = 10; console.log(age++ + 10) // 20 区别 前置后置单独使用 效果一样 结合运算就不一样了 var e = 10; var f = e++ + ++e; // 22 e++ 先运算后自增 e++为10 因为运算是自左往右 到了++e e因为e++后自增变为11 而++e先自增后运算 所以++e为12

比较运算符

< > <= >= ==(判断号 会转型) !=不等号 === !==(全等 值和数据类型都一致) 37 == '37' // true 37 === '37' // false = 赋值 把右边给左边 == 判断 判断俩边的值是否相等(有隐式转换) === 全等 判断俩边的值和数据类型是否完全相同

逻辑运算符

&& 逻辑与 简称与 and true && false 与 都为真才是真 || 逻辑或 简称或 or true || false 或 有一个真就是真 ！ 逻辑非 简称非 not !true 非 反着来 短路运算(逻辑中断) 短路运算永远返回能决定表达式真假的那个表达式 原理(当有多个表达式(值)时 左边的表达式值可以确定时 就不再继续运算右边的表达式值) 逻辑与 短路运算 语法 表达式1 && 表达式2 遇真往后走 遇假选假 如果第一个表达式值为真 返回表达式2 如果第一个表达式值为假 返回表达式1 逻辑或短路运算 返回值为真(自己总结) 如果第一个表达式为真 返回表达式1 如果第一个表达式为假 返回表达式2 var num = 0; console.log(123 || num++) // 逻辑或短路运算 第一个为真 返回值123 后面的不执行 console.log(num) // 0 数字除0外 其他数字都是true

赋值运算符

= 直接赋值 += -= *= /= %= var age = 6; age += 5; // age = age + 5; age -= 5; // age = age - 5; age *= 8; // age = age * 8;

优先级

1 小括号 () 2 一元运算符 ++ -- ！ 3 算数运算符 先*/% 后+- 4 关系运算符 > >= < <= 5 相等运算符 == != === !== 6 逻辑运算符 先&& 后|| 7 赋值运算符 = 8 逗号运算符 , 计算时 用逻辑运算符分隔开
