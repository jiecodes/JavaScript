函数 封装了一段可以被重复使用的代码块 目的:让大量的代码重复使用 函数的封装 把一个或多个功能通过函数的方式封装起来 对外只提供一个简单的函数接口 函数使用 声明函数 function 函数名() { 函数体 } 注意点 function 声明函数的关键字 全部小写 函数是做某件事 函数名一般是动词 sayHi 函数不调用自己不执行 调用函数 函数不调用 自己不执行(所以一个函数 从调用函数开始执行) 函数名(); 函数参数 作用:在函数内部某些值不能固定 可以通过参数在调用函数时传递不同的值进去 形参:形参是接收实参的 形参类似于一个变量(不需要声明的变量) 实参:实参个数可以有 可以没有 利用函数的参数实现函数重复不同的代码 function 函数名(形参1,形参2...) { // 声明函数的小括号里面是形参(形式上的参数) } 函数名(实参1,实参2...) // 函数调用的小括号里面是实参(实际的参数) 实参个数大于形参个数 会取到形参的个数(多余的实参被扔掉) 实参个数小于形参个数 多余的形参定义为undefined 最终结果为NaN(数字型+undefined=NaN) 函数的返回值 return语句 function 函数名() { return 需要返回的结果; } 函数名(); 函数只是做某件事 最终处理完的结果返回给函数的调用者 函数名() 通过return实现 只要函数遇到return 就把结果返回给函数的调用者 函数名()=return后面的结果 return 终止函数 return 下一行(后面的)代码不会被执行 function fn(num1, num2) { return num1, num2 alert('不会执行') } console.log(fn(1, 3)) return 只能返回一个值 function fn(num1, num2) { return num1, num2 // 返回的结果是最后一个值 } console.log(fn(1, 3)); // 结果为3 任意俩数 加减乘除 结果 return返回多个值 以数组的形式 function getResult(num1, num2) { return [num1 + num2,num1 - num2,num1 * num2,num1 / num2] } var result = getResult(1, 3) // 返回一个数组 console.log(result) 函数没有 return 返回 undefined function fn(num1 num2) { } console.log(fn(1, 3)) 函数 function 函数名(形参1, 形参2) { 函数体; return 返回值; } 函数名(实参1, 实参2) arguments(参数) 只有函数才有 arguments 而且每个函数都内置好了arguments arguments使用:里面存储里所有用户传递过来的实参 不确定用户传过来多少个值时,可以用arguments function fn() { console.log(arguments) // 里面存储了所有传递过来的参数 arguments = [3,6,1] console.log(arguments.length) console.log(arguments[2]) // 2表示索引号 // 把arguments里面的所有实参遍历出来 for (var i = 0;i < arguments.length;i++) { console.log(arguments[i]); } } fn(3, 6, 1) arguments 伪数组 并不是真正意思上的数组 具有数组的length属性 可以判断数组长度 按照索引的方式进行存储 没有真正数组的一些方法 pop() push() 函数声明的两种方式 1 利用函数关键字自定义函数(命名函数) function fn() { } fn(); 2 函数表达式(匿名函数) 变量名 = function() {}叫函数表达式 var 变量名 = function() {}; var fn = function(aru) { console.log('我是函数') console.log(aru) } fn('你好');
