顺序结构 从上往下执行代码 分支结构 由上至下执行代码过程中 根据条件不同 执行不同代码 得到不同结果 if语句 语句结构 执行思路:如果条件表达式为真 执行语句 为假 跳过执行语句 if (条件表达式) { // 执行语句 } if (3 > 5) { alert('你好') } if else 双分支语句 多选一 最后只能执行一句语句 语句结构 执行思路:如果表达式结果为真 执行语句1 结果为假 执行语句2 if (条件表达式) { // 执行语句1 } else { // 执行语句2 } if else if 多分支语句 语句结构 执行思路:如果满足条件1 则执行语句1 退出if循环 不满足条件1 执行条件2 如果都不满足 执行else if (条件表达式1) { 语句1 } else if (条件表达式2) { 语句2 } else if (条件表达式3) { 语句3 } else { 最后的语句 } 三元表达式 由三元运算符组成的式子 简化版的if else ++num 一元运算符 3 + 5 二元表达式 ? : 三元表达式 语法结构 执行思路:如果条件表达式为真 执行表达式1 如果为假 执行表达式2 条件表达式 ? 表达式1 : 表达式2 var num = 10; var result = num > 5 ? '是的' : '不是的'; console.log(result) switch语句 多分支语句 也可以实现多选一 语法结构 switch(转换 开关) case(例子 选项) 执行思路:用表达式的值和case后面的值匹配 如果匹配上 就执行case里面的语句 都没匹配上 执行default语句 switch (表达式) { case value1: 执行语句1 break; case value2: 执行语句2 break; default: 执行最后语句; } 表达式和value1里面的值匹配的时候 关系是全等(值 数据类型) === break 如果当前case里面没有break 不会退出switch 继续执行下一个case if else 与 switch case 区别 switch case 通常处理比较确定值的情况 if else 用于范围的判断 switch 进行条件判断后 直接执行条件语句 if else 有几个条件 判断几次 分支比较少时 if else 效率比较高 分支比较多时 switch case 效率比较高

循环结构

目的: 重复执行某些代码 通常和计数有关系 for循环 for (初始化变量; 条件表达式; 操作表达式) { // 循环体 } 初始变化量 用var声明一个普通变量 通常用作计数器使用 条件表达式 用来决定每一次循环是否继续执行 这是终止条件 操作表达式 每次循环最后执行的代码 用于计数器变量进行更新(递增 递减) 主要目的:更新初始变量 for (var i = 1; i <= 100; i++) { console.log('hello word') } for 循环执行过程 1 首先执行计数器变量 var i = 1, 这句话在for里面只执行一次 2 i <= 100 判断是否满足条件 如果满足 执行循环体 不满足 直接退出循环 3 i++ 最后执行 i++是单独写的代码 递增 递减 第一轮结束 4 接着执行 i <= 100; 满足条件 执行循环体 不满足条件 退出循环 第二轮结束 一直循环下去 断点调试 在Chrome f12 调试 如果不出结果 用 f5 双重for循环 for (外层初始化变量;外层条件表达式;外层操作表达式) { for (内层初始化变量;内层条件表达式;内层操作表达式) // 执行语句 } 1 把里层循环看做是外层循环的语句 2 外层循环一次 里层的循环执行全部 while循环 while 当...的时候 执行思路:当条件表达式为true 执行循环体 否则 退出循环体 while (条件表达式) { // 循环体 } var num = 1; // 计数器 while (num <= 100) { console.log('你好') num++; // 不加会进入死循环 } 执行过程:先执行num=1 接着num<=100 结果为true 执行循环体 为false 退出循环 再执行num++ 再到num<=100 do while循环 do { // 循环体 } while (条件表达式) 执行思路:先执行循环体 再进行判断 如果条件为真 则执行循环体 否则 退出循环体 var i = 1; // 计数器 do { console.log('你好') i++; // 操作表达式 } while (i <= 100) // 条件判断 continue break 关键字 continue(继续) 退出当前循环 继续执行剩余次数循环 for (var i = 1;i <= 5;i++) { if (i == 3) { continue // 退出当前循环 不再执行下面代码 跳到i++ } console.log('第'+ i + '个') } 执行思路:i=1 i<=5 判断i是否等于3 =3 执行continue 退出循环 接着执行i++ 不等于3 执行循环体 然后再执行i++ break(打破) 退出整个循环 for (var i = 1;i <= 5;i++) { if (i == 3) { break; } console.log('第' + i + '个') } 命名规范 变量 函数的命名必须有意义 变量一般用名称 函数一般用动词 操作符 左右俩侧保留一个空格 单行注释规范 注释的后面写一个空格 小括号左右俩侧包含一个空格
